Features:

Home Page: Displays featured playlists and recommended music.

Search Functionality: Allows users to search for songs, artists, and albums.

Playlist Display: Shows user-created and recommended playlists with album artwork.

CSS: Styling and layout of the web pages.

Bootstrap: Ensures responsiveness and adds UI components like the navbar, cards, and grid system for layout.
